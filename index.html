Database Setup

Connect a free Supabase database so volunteer sign-ups persist for everyone, across devices, with no server cost.

1. Go to supabase.com → create a free account & new project
2. In SQL Editor, run:
create table schedule (id text primary key, data jsonb);
3. Go to Project Settings → API → copy your URL and anon key
4. Paste them below and click Connect
